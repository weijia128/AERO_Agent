## 二、信息收集 - 必问内容

收到漏油报告后，系统会按以下顺序收集关键信息：

| 序号 | 询问内容 | 获取信息 | 示例回答 |
|:---:|---------|---------|---------|
| 1 | "报告你机号" | 航班号 | "南航3456" |
| 2 | "具体位置和油液类型？" | 停机位/滑行道 + 燃油/液压油/滑油 | "501机位，燃油" |
| 3 | "发动机状态？是否持续滴漏？" | 运转/关车 + 是/否 | "发动机运转，持续滴漏" |

**重要提示：**
- ✅ 以上5项信息收集完成后，系统**立即进行风险评估**
- ✅ 获知航班号后，用航班号称呼（如"南航3456，请告知..."）
- ❌ 不要用航班计划停机位替代实际事发位置

## 三、风险等级判定标准

系统根据收集的信息，自动判定风险等级：

### 风险等级说明

| 等级 | 风险程度 | 典型场景 | 响应要求 |
|:---:|:-------:|---------|---------|
| **R4** | 🔴 严重 | 燃油+发动机运转 / 燃油+持续泄漏 | 3分钟内消防响应 |
| **R3** | 🟠 高 | 燃油泄漏 / 液压油+发动机运转 | 消防待命+机务响应 |
| **R2** | 🟡 中 | 液压油常规泄漏 | 部门待命+吸附处理 |
| **R1** | 🟢 低 | 滑油少量泄漏 | 清洗部门处理 |

### 快速判定表

| 油液类型 | 发动机运转 | 持续泄漏 | 大面积 | → 风险等级 |
|:-------:|:--------:|:-------:|:-----:|:---------:|
| 燃油 | ✓ | ✓ | - | **R4** |
| 燃油 | ✓ | - | - | **R4** |
| 燃油 | - | ✓ | - | **R4** |
| 燃油 | - | - | ✓ | **R4** |
| 燃油 | - | - | - | R3 |
| 液压油 | ✓ | ✓ | - | R3 |
| 液压油 | - | ✓ | - | R3 |
| 液压油 | - | - | - | R2 |
| 滑油 | - | ✓ | - | R2 |
| 滑油 | - | - | - | R1 |

**核心原则：燃油风险最高，发动机运转时任何燃油泄漏都是R4**

## 五、综合分析内容

完成风险评估后，系统自动进行综合影响分析，包括：

### 1. 清理时间预估

| 油液类型 | 小面积(<1m²) | 中面积(1-5m²) | 大面积(>5m²) |
|:-------:|:-----------:|:------------:|:-----------:|
| 燃油 | 30分钟 | 60分钟 | 90分钟 |
| 液压油 | 20分钟 | 45分钟 | 75分钟 |
| 滑油 | 15分钟 | 30分钟 | 60分钟 |

*注：恶劣天气（大风/低温/大雨）会延长清理时间*

### 2. 空间影响范围

系统自动计算受影响区域：
- **受影响机位**：根据位置和风险等级计算邻近机位
- **受影响滑行道**：判断是否影响滑行路径
- **受影响跑道**：R4/R3等级会评估跑道影响

### 3. 航班影响预测

基于清理时间窗口，预测受影响的航班：
- 进港航班延误预警
- 出港航班推迟预警
- 滑行路径受阻航班

### 4. 风险场景分析

| 风险类型 | 说明 | 示例 |
|---------|-----|-----|
| 安全风险 | 人员/设备安全威胁 | 燃油蒸发+高温=火灾风险 |
| 运行风险 | 机场运行效率影响 | 跑道关闭导致航班延误 |
| 扩散风险 | 泄漏扩大可能性 | 大风+持续泄漏=范围扩大 |
| 连锁风险 | 引发次生事件 | 地面设备打滑事故 |

## 六、处置流程时间线

```
T+0分     收到报告，开始信息收集
T+2分     完成关键信息收集
T+3分     风险评估完成，通知相应部门
T+5分     综合分析完成，制定处置方案
T+8分     消防/清污人员到达现场（R4）

```

## 七、常见问答

**Q1: 为什么燃油泄漏风险最高？**
> 燃油闪点低（约38°C），易挥发形成可燃气体，遇火星即可引燃。发动机高温部件是主要点火源。

**Q2: 液压油和滑油有什么区别？**
> 液压油用于飞机操控系统，压力高（3000psi+），泄漏时可能高压喷射。滑油用于发动机润滑，通常低压泄漏。

**Q3: 为什么要单独确认事发位置？**
> 航班计划停机位可能与实际位置不同（如滑行中发现泄漏）。准确位置决定隔离范围和受影响区域。

**Q4: R3和R4都要通知消防，有什么区别？**
> R4要求消防3分钟内到场准备处置；R3是消防待命，准备响应但不立即出动。

**Q5: 系统如何计算受影响航班？**
> 根据事发位置周边机位的航班计划，结合清理时间窗口（清理时间+30分钟缓冲），筛选时间窗口内的进出港航班。
