> **适用范围**：机坪航空器漏油、油污、渗漏等特情事件的识别、处置与闭环记录
> **目的**：规范现场处置流程，确保运行安全、合规与审计可追溯

## 模板变量说明

| 变量 | 说明 | 默认值 |
|-----|------|-------|
| `{event_id}` | 事件编号 TQCZ-YYYYMMDD-NNN | 自动生成 |
| `{flight_number}` | 航班号或航空器注册号 | 必填 |
| `{event_time}` | 事件触发时间 | 必填 |
| `{report_method}` | 上报方式 | 巡查 |
| `{reporter}` | 报告人 | 待填写 |
| `{location}` | 发现位置 | 必填 |
| `{oil_type}` | 油液类型 | 不明 |
| `{is_continuous}` | 是否持续滴漏 | 否 |
| `{engine_status}` | 发动机/APU状态 | 关闭 |
| `{leak_area}` | 泄漏面积 | 待评估 |
| `{leak_pattern}` | 漏油形态 | 滴漏 |
| `{weather}` | 气象条件 | 晴 |
| `{check}` | 复选框状态 | ☐ 或 ☑ |
